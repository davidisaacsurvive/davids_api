# SETTING UP THE CODE ON YOUR MACHINE 
### STEP 1
<p>Clone the repository</p>

### STEP 2 
<p>Create A virtual Environment</p>

### STEP 3 
<p>RUN THE COMMAND AFTER ACTIVATING YOUR VIRTUAL ENVIRONMENT </p>
<code> pip install -r requirements.txt </code>

### STEP 4
<p> Create a <code>.env</code> file following the <code>.env.sample</code> format </p>

### STEP 5
<p> Run your development server</p>
